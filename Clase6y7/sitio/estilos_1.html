¿Como podemos implementar estilos?

Existen tres maneras, pero solo una es buena paractica a la hora de desarrollar

Estilos en linea

Cuando utilizamos el atributo «style» dento de la etiqueta de HTML

Estilos internos

Dentro de la etiqueta «head» del documento (metadatos) utilizando la etiqueta «style»

Estilos externos

Manera recomendada.

Dentro de la etiqueta «head» agregamos una etiqueta «link» que conecta con nuestro script de CSS

De acuerdo al documento de ejemplo y teniendo en cuenta la casacda, evaluar como se implemento jererquicamente cada estilo.
